Trolley - 一个轻量级高性能的.NET ORM框架
========================================

## 介绍
一款轻量级高性能的ORM，可以不用写SQL语句，支持分页，事务，多结果集，多命令执行，模型映射，导航属性，CTE，批量插入、批量更新，分库分表等，各种复杂SQL场景等。
